# 🎵 Music Store Business Analysis using SQL

## 📌 Project Overview

This project analyzes the Chinook Music Store database using PostgreSQL to answer business questions related to customer behavior, sales performance, music preferences, and revenue trends.

## 🎯 Objectives

* Analyze customer spending patterns.
* Identify top-performing artists and genres.
* Explore revenue across different countries.
* Generate business insights for decision-making.

## 🛠 Tools Used

* PostgreSQL
* PgAdmin
* VS Code

## 📊 SQL Skills Demonstrated

* JOINs
* GROUP BY
* Aggregate Functions
* Subqueries
* CTEs
* CASE
* COALESCE
* Window Functions
* DATE_TRUNC
* LAG
* DENSE_RANK

## 📋 Key Business Questions

* Who are the best customers?
* Which countries generate the most revenue?
* Which genres and artists perform best?
* What are the monthly revenue trends?
* How can customers be segmented based on spending?

## 📈 Key Insights

* High-revenue countries are important growth markets.
* Popular music genres drive a significant share of sales.
* High-value customers contribute substantially to overall revenue.
